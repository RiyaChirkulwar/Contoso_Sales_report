# 📊 Contoso Sales Analytics Dashboard

## Project Overview

**Contoso Sales Analytics Dashboard** is a comprehensive Business Intelligence solution developed using Power BI. This interactive dashboard analyzes Contoso's sales performance, profitability, product metrics, and regional trends to enable data-driven decision-making.

---

## 🎯 Purpose & Objectives

### Target Audience
- Business analysts
- Sales managers
- Executive leadership
- Decision-makers requiring organizational performance insights

### Primary Goals
- Monitor key business KPIs in real-time
- Analyze sales and profit performance across multiple dimensions
- Track temporal trends (monthly, quarterly, yearly comparisons)
- Evaluate product and brand profitability
- Compare regional and store-level performance
- Support strategic business decision-making through visual analytics

---

## 🛠️ Technology Stack

| Component | Tool/Technology |
|-----------|-----------------|
| **Primary Platform** | Power BI Desktop |
| **Data Processing** | Power Query (ETL) |
| **Calculations & Logic** | DAX (Data Analysis Expressions) |
| **Data Architecture** | Star Schema Data Modeling |
| **Visualizations** | Interactive charts, KPI cards, waterfall charts, tables |

- `.pbix` – Power BI development file

## 📂 Data Source

**Source:** Kaggle – Contoso Sales Dataset

### Dataset Components
- Sales transactions and order details
- Product information and categories
- Customer demographics and purchase history
- Geographic data (country, state, city)
- Store locations and operational data
- Profit and cost metrics
- Complete date hierarchy for time-based analysis

### Data Preparation
- Duplicate record removal
- Missing value handling
- Column standardization and renaming
- Data type optimization
- Date hierarchy creation

---

## 📊 Business Problem & Solution

### Challenges Addressed
Retail organizations process massive volumes of transactional data daily. Key business questions require quick, accurate answers:
- Which regions generate the highest profit?
- Which products drive maximum revenue?
- How has profitability changed over time?
- What are the sales and return trends?
- Which locations require strategic attention?

### Solution Delivered
A centralized analytics platform providing instant answers to critical business questions, eliminating manual analysis and enabling faster decision-making.

## 📈 Key Features & Metrics

### KPI Dashboard (Top Section)
- **Return Quantity:** 29K
- **Net Quantity Sold:** 3M
- **Total Sales Amount:** $697M
- **Discount Amount:** $12.30M
- **Return Amount:** $8.6M
- **Cost of Goods Sold:** $297.06M
- **Net Profit:** $379M

### Interactive Filter Panel
Users can dynamically filter by:
- Channel
- Region/Country
- State/Province
- City
- Store
- Product Category
- Product Subcategory

### Core Visualizations

#### 1. **Net Profit by Month & Year (Combination Chart)**
- Current year vs. previous year comparison
- Year-over-Year (YoY) profit percentage analysis
- **Benefits:** Monthly trend monitoring, seasonal pattern identification, growth opportunity detection

#### 2. **Net Profit Running Total by Date Hierarchy**
- Cumulative profit visualization over time
- **Benefits:** Long-term growth tracking, quarterly/annual progression analysis

#### 3. **Year-over-Year Net Profit by Product Brand (Waterfall Chart)**
- Brand-level contribution to overall profit
- **Benefits:** Top brand identification, portfolio optimization insights

#### 4. **Geography & Store Details Matrix**
- Performance breakdown across continents, countries, states, and cities
- **Key Metrics:** Total Sales, Net Quantity, Net Profit, Order Count, Average Order Value
- **Benefits:** Regional benchmarking, geographic performance analysis, store-level profitability
